Sign Up

Example (Success):

$ curl \
    -d 'email=...&password=...&name=...' \
    -X 'POST' \
    http://localhost:5000/sign-up
200 OK
{
    "email": "...",
    "id": "...",
    "name": "...",
    "token": "..." # expires in 24 hours
}

Example (Failure):

$ curl \
    -d 'email=&password=&name=' \
    -X 'POST' \
    http://localhost:5000/sign-up
400 Bad Request
{
    "exception": "..."
}

Parameters:

  1. email
    • mandatory
  2. password
    • mandatory
  3. name
    • mandatory

Dashboard

Example:

$ curl \
    -H 'Token: ...' \
    -X 'GET' \
    http://localhost:5000/dashboard?date=...
200 OK
{
    "average_speed": "...", # in meters per second
    "total_distance": "..." # in meters
    "total_time": "...",     # in seconds
}

Parameters:

  1. date
    • optional
    • format: YYYY-MM-DD (report will be limited to a period between date - 7 days and date)

List Entries

Example:

$ curl \
    -d 'dates_from=...&dates_to=...&order_by_column=...&order_by_direction=...&offset=...&limit=...' \
    -H 'Token: ...' \
    -X 'GET' \
    http://localhost:5000/entry
200 OK
{
    "items": [
        {
            "date": "...",
            "distance": "...", # in meters
            "id": "...",
            "time": "..."      # in seconds
        },
        ...
        {
            "date": "...",
            "distance": "...", # in meters
            "id": "...",
            "time": "..."      # in seconds
        }
    ],
    "total": "..."
}

Parameters:

  1. dates_from
    • optional
    • format: YYYY-MM-DD
  2. dates_to
    • optional
    • format: YYYY-MM-DD
  3. order_by_column
    • optional
    • default: id
    • choices: id, date, distance, time
  4. order_by_direction
    • optional
    • default: desc
    • choices: asc, desc
  5. offset
    • optional
    • format: a valid integer (will be coerced)
    • default: 0
  6. limit
    • optional
    • format: a valid integer (will be coerced)
    • default: 10
    • maximum: 50

Get Entry

Example (Success):

$ curl \
    -H 'Token: ...' \
    -X 'GET' \
    http://localhost:5000/entry/<id>
200 OK
{
    "date": "...",
    "distance": "...", # in meters
    "id": "...",
    "time": "..."      # in seconds
}

Example (Failure):

$ curl \
    -H 'Token: ...' \
    -X 'GET' \
    http://localhost:5000/entry/<id>
400 Bad Request
{
    "exception": "..."
}

Parameters:

  1. date
    • mandatory
    • format: YYYY-MM-DD (must be today or older; i.e. you cannot add entries in the future)
  2. distance (in meters)
    • mandatory
    • format: a valid double (will be coerced)
  3. time (in seconds)
    • mandatory
    • format: a valid double (will be coerced)
